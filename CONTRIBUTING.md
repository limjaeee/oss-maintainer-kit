# Contributing

Thanks for considering a contribution.

## Development

```bash
npm install
npm test
npm run test:coverage
```

## Pull requests

Please include:

- The maintainer workflow your change improves.
- Tests for new behavior.
- Compatibility or security notes when relevant.

## Project direction

This project favors small, auditable automation over broad agent behavior. A useful change should make maintainer review, triage, or release work more concrete.
